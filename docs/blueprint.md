# **App Name**: LSPP do INSS

## Core Features:

- Pedido de Benefícios: Permitir que usuários solicitem benefícios do INSS diretamente pelo aplicativo, fornecendo os dados necessários e anexando documentos digitalizados.
- Consulta de Benefícios: Possibilitar a consulta de informações detalhadas sobre todos os benefícios disponíveis no INSS.
- Cadastro de Dados: Permitir que o usuário insira e salve seus dados pessoais completos para facilitar o preenchimento de formulários futuros.
- Upload de Documentos: Permitir o upload de fotos e outros documentos necessários para a solicitação de benefícios.
- Monitoramento do Status: Acompanhar o status das solicitações de benefícios enviadas, com notificações sobre atualizações, em análise, exigência, compareça presencialmente.

## Style Guidelines:

- Primary color: Blue (#2979FF), chosen to convey trust and reliability, characteristics expected from an application related to legal government services.
- Background color: Light gray (#F0F4F7), providing a neutral and clean backdrop that emphasizes legibility and reduces distractions.
- Accent color: Light blue (#53A4FF), enhancing user interface elements with a visually distinct color.
- Body and headline font: 'Inter', a sans-serif font known for its clean and modern appearance, ensuring readability and clarity across all text elements. This font ensures great legibility on various screen sizes and resolutions.
- Utilize icons que remetem a benefícios, documentos e status.
- Layout clean and intuitive, prioritizing ease of navigation.
- Subtle animations for a better UX.
